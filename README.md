我和我的妻子 [Vanessa](https://github.com/Vanessa219) 从 2009 年开始编写开源软件，其中我们一直在维护的 [Solo](https://github.com/88250/solo) 项目到现在已经有 10 年了。我们的创作领域主要围绕博客和社区系统开展，我们一直在努力实现 [B3log 构思 - 分布式社区网络](https://hacpai.com/article/1546941897596)。

从 2018 年开始，我和 Vanessa 从就职的公司离职，正式开始了“全职做开源、自由职业者”的创业生涯。我们建立了一家公司，主要产品是 Sym 社区系统，它有完全开源免费的[社区版](https://github.com/88250/symphony)。同时，我们也运营着一个超过 5 万用户的论坛[黑客派](https://hacpai.com)，作为 B3log 分布式社区网络的社区端节点。

最近我们开启了新项目 [Vditor](https://github.com/Vanessa219/vditor)，这是一款浏览器端的 Markdown 编辑器，它有三种编辑模式：传统的分屏编辑预览、展开标记符的即时渲染和类似富文本编辑的所见即所得。目前这三个模式已经基本可用，欢迎大家来试用和反馈。

对我和 Vanessa 来说，开源已经不仅仅只是一种爱好，更是一种生活方式，我们对这条“开源生活”之路的未来充满了信心。希望在这条路上我们能通过开源软件帮助到其他人，同时其他人也能帮助到我们。

开源连接你我，开源构建未来，让我们一起走进开源的世界！
